# Hello World Club 🚀

**Hello World Club** é um projeto de plataforma para **match de desenvolvedores interessados em Pair Programming** feito por Vinicíus A Leal com minha participação como Front-end Dev.

A ideia é conectar programadores com interesses e tecnologias em comum para praticar, aprender e desenvolver projetos juntos.

O projeto ainda está em desenvolvimento (MVP em construção), mas já possui **landing page funcional** e planejamento técnico da arquitetura.

---

## 💡 Objetivo

Criar um sistema que:

- Conecte desenvolvedores para **pair programming**
- Sugira parceiros automaticamente através de **algoritmo de match**
- Permita **aceitar ou recusar matches**
- Envie **notificações por Email ou Discord**
- Apresente **métricas de uso para administradores**

## 🧠 Como funciona o Match

O algoritmo calcula compatibilidade entre desenvolvedores considerando:

- Interesse no projeto
- Nível de senioridade
- Experiência
- Tecnologias em comum
- Disponibilidade de horário
